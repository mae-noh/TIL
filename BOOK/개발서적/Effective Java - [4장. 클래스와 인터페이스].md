### item 15. 클래스와 멤버의 접근 권한을 최소화하라.*
- 모든 클래스와 멤버의 접근성을 가능한 좁혀야 한다.
- 멤버 접근자
  - `private` 멤버를 선언한 클래스에서만 접근 가능
  - `package-private` 멤버가 속한 패키지 안의 모든 클래스에서 접근 가능, 접근제한자를 명시하지 않는 경우 기본 접근 수준
  - `protected` package-private + 멤버 하위 클래스에서 접근 가능
  - `public` 모든 곳에서 접근 가능
